Basic Overview

This section is to demonstrate how a User may use this App when it has been deployed
(The User is not running the code on their local machine).

1) If not already on the "Home" Page, click on Home Link at the top of the page

2) Click on the button

-- Voila! You can either Play the MIDI File or Download it within your browser!
Feel free to repeat this with the button!!

How to Play MIDI File Locally by Your System Type

Operating System Needed Software Instructions
Windows (Windows 10 64-Bit) Windows Media Player (Included) Simply Open the MIDI_[your_template_id].mid file to Play
Android Mobile (Version 12) MIDI Player (Google Play Store) Simply Open MIDI Player and hit Play
